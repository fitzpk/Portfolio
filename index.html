Web Scraping with Selenium

Through the utilization of the Selenium library, these python scripts enable the user to interact with data sources such as NSERC's research awards database and Shanghai Ranking's online tables, and ultimately extract detailed data that is otherwise not exportable.



Song Recommendation Engine

An item-content similarity recommendation engine constructed using more than 10,000 songs and the Turicreate library in Python. By extracting and comparing audio metrics from songs, similar items are recommended to users based on their most recent playlist.



Predicting Golf Course Turf Species

Using a multi-output classification model, this project predicts the turf species of golf course greens and fairways. From web scraping data to creating a hyper-tuned model, this project was completed almost entirely in Python.

Local Business Analysis Application

Online map, driven by the JavaScript-based Leaflet API, that serves as a resource for community members to identify local businesses, assess competition, and analyze the spatial distribution within their town or city.



Canadian Diabetes Association - Routing System

Constructed using ESRI App Builder, this GIS application provides routing analysis and optimal route plans for vehicle fleets sent out by the CDA to collect donations at their designated boxes.



Live GeoTwitter App

This product uses the Tweepy Python module to stream tweets, export them to a JSON file, and manipulate them into GeoJSON data. Data is then pushed to Github in timed intervals, where it is made available in a custom web map constructed using Leaflet, JavaScript, and jQuery.

Locating High Potential Markets for Shell Canada Stations

In this process, threshold area analysis, find similar sites analysis, and huff modelling in ArcGIS are combined to provide the analytical insights needed to make informed decisions regarding franchise expansion

Modelling Least-Cost Habitat Corridors for White-Tailed Deer

Using a combination of Spatial Analyst tools and map algebra techniques within ArcGIS Model Builder, this project discerns a least-cost corridor between two potential white-tailed deer habitats in any location.
