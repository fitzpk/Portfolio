Local Business Analysis Application

Online map, driven by the JavaScript-based Leaflet API, that serves as a resource for community members to identify local businesses, assess competition, and analyze the spatial distribution within their town or city.



Canadian Diabetes Association - Routing System

Constructed using ESRI App Builder, this GIS application provides routing analysis and optimal route plans for vehicle fleets sent out by the CDA to collect donations at their designated boxes.



Live GeoTwitter App

This product uses the Tweepy Python module to stream tweets, export them to a JSON file, and manipulate them into GeoJSON data. Data is then pushed to Github in timed intervals (currently 10 mins), where it is made available in a custom web map constructed using Leaflet, JavaScript, and jQuery.

Database Automation with PL/SQL Scripting

Through Oracle PL/SQL scripting, this program creates a relational data warehouse designed to store municipal property data. It then retrieves data from a shared server, structures the data, and finally imports it into designated tables

Predictive Modelling Patterns in Medical Expenditure

Through exploratory regression and ordinary least squares modelling, this project mines medical spending data from the Canadian census, identifies patterns and relationships, and predicts future trends in medical spending for Canadians.

Python Tkinter Apps for GeoJSON Analysis

Utilizing Tkinter GUIs, these two apps process GeoJSON data to perform geometric calculations. The first application is a simple polygon area and perimeter calculator, while the second application is a point-in-polygon analyzer.

Modelling Least-Cost Habitat Corridors for White-Tailed Deer

Using a combination of Spatial Analyst tools and map algebra techniques within ArcGIS Model Builder, this project discerns a least-cost corridor between two potential white-tailed deer habitats in any location.

Locating High Potential Markets for Shell Canada Stations

In this process, threshold area analysis, find similar sites analysis, and huff modelling in ArcGIS are combined to provide the analytical insights needed to make informed decisions regarding franchise expansion

Modelling Watershed Analytics

Utilizing hydrological and surficial tools within ArcGIS Model Builder, this process allows conservation authorities and county-level governments to effectively assess the hydrological characteristics and composition of the watersheds they govern.
